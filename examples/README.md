# Pixel Matrix // Examples

* [program-rand-lines.py](./program-rand-lines.py) -- A Pixel Matrix Program, defined as a sub-class of `Program`, that draws random colored lines on the screen.
* [function-rand-dots.py](./function-rand-dots.py) -- A Pixel Matrix Program, defined as a single function and executed with the `Program.exec_func` class method, that draws random colored dots on the screen.
* [standalone-rand-dots.py](./standalone-rand-dots.py) -- Code that uses the `PixelMatrix` class directly (defnining it's own event loop) to draw random colored dots on the screen.
